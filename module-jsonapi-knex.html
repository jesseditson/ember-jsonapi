jsonapi-knex

(require("jsonapi-knex"))(db, schemas, tables) → {object}

jsonapi-express operations for knex This will fill in all the required operations, and return an object ready for decoration. You may add a transforms dictionary and/or an authorize middleware function to this object before passing to jsonapi-express.
Parameters:
Name Type Description
db knex A preconfigured knex instance
schemas object A dictionary of schemas
tables object A dictionary of name:table mappings, for specifying what a schemas' table is named. If a schema is unspecified, the table name will be assumed to match the schema name.

An operations dictionary to pass to jsonapi-express
